# KantinKu v2.0
Sistem Pemesanan Makanan Digital Kampus yang modern, cepat, dan efisien.

## Fitur Utama
- **Standardisasi UI**: Menggunakan Lucide React untuk ikonografi yang premium dan konsisten.
- **Status Operasional Otomatis**: Kedai berubah status menjadi Buka/Tutup secara real-time berdasarkan jam operasional.
- **Manajemen Multi-Role**: Dukungan untuk Admin, Merchant (Pemilik Kedai), dan Customer (Mahasiswa).
- **Interactive Reviews**: Sistem ulasan bintang interaktif untuk feedback pelanggan.
- **KantinPay Integration**: Sistem pembayaran digital internal kampus.

## Teknologi
- **Frontend**: React.js, Vite, Lucide React, Vanilla CSS.
- **Backend**: Node.js, Express, MySQL.
- **Auth**: JWT (JSON Web Token).
